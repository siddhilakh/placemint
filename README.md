# PlaceMint

AI-powered placement companion for Indian engineering students.

## What it does

PlaceMint analyses your resume against your profile — branch, CGPA, college tier — and gives you an ATS score, a list of roles you're realistically eligible for, and a specific gap report with actionable fixes. Built for the Indian campus hiring context, not generic US tech hiring.

## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS
- PostgreSQL + Prisma (Week 3)
- Gemini API (Week 4)

## Week 1 — What was built

- Project setup with Next.js, TypeScript, Tailwind
- Full folder structure for all 8 weeks
- Landing page — Navbar, Hero, Features, How it works, Footer
- Profile form at /profile with all 5 fields
- Core TypeScript types — StudentProfile, ResumeAnalysis, RoleSuggestion, ResumeGap

## What confused me

Tailwind's responsive prefix (md:) took a moment to understand — it means "apply this class only on medium screens and above." Once that clicked, building the responsive navbar made sense.

## What I'd do differently

I'd commit after each individual section instead of grouping Day 3 and Day 4 into one commit. Smaller commits make it easier to track exactly what changed and when.
